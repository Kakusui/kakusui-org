Okisouchi

Welcome to Okisouchi, by using this service you agree to the terms of service and privacy policy.


Okisouchi is a robust, open-source tool for automating the transfer of files from Google Drive to designated locations, streamlining file management and organization with user-configured settings while ensuring data privacy and adherence to GPLv3 open-source licensing.


To Use:


The whole process is detailed further on where the source code is located here but put simply:


Simply download the source code, set up your LocalConfig file, and configure the following paths:


  • blacklist.txt (folder names to ignore when merging directories)
  • file_names.txt (files to import from)
  • folder_ids.txt (gfolder id's to retrieve content from)
  • folder_names.txt (those gfolder names)
  • target_location.txt (where to put the files)


Once this is completed, you can simply run the run.py file and log in through Google, and your transfer will complete.


We recommend that you read the full README file on the GitHub repository to understand the full process at this link.
